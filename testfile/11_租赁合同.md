## 五、主要权利义务条款

| 条款 | 内容 |
|------|------|
| 装修条款 | 承租方有权对租赁物业进行装修，装修方案须经出租方书面同意 |
| 转租条款 | 未经出租方书面同意，不得转租或分租 |
| 优先购买权 | 合同到期前，如物业出售，承租方有优先购买权 |
| 提前解约 | 任一方提前3个月书面通知，可解除合同（押金不退） |
| 维修责任 | 物业主体结构由出租方负责，内部装修及设施由承租方负责 |
| 动产抵押 | 租赁期间，承租方不得将价值超过50万元的设备抵押给第三方 |
