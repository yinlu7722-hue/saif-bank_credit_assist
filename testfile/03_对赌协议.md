# 对赌协议条款摘录
# 《关于深圳市辰星智联科技有限公司之增资协议》补充协议

---

## 协议基本信息

| 项目 | 内容 |
|------|------|
| 协议名称 | 关于深圳市辰星智联科技有限公司之增资协议补充协议（对赌条款） |
| 签署日期 | 2021年6月15日 |
| 投资方 | 深圳远致中创天使投资基金（有限合伙） |
| 融资轮次 | A轮 |
| 增资金额 | 人民币1,500万元 |
| 获得股权 | 10%（投后） |
| 投后估值 | 1.5亿元 |

---

## 第5.1条 业绩承诺

| 会计年度 | 营业收入最低承诺 | 净利润最低承诺 | 实际完成情况 |
|---------|---------------|--------------|------------|
| 2021年 | 5,000万元 | 400万元 | 营收5,200万，净利润420万 ✓ |
| 2022年 | 8,000万元 | 650万元 | 营收8,500万，净利润680万 ✓ |
| 2023年 | 12,000万元 | 900万元 | 营收12,000万，净利润920万 ✓ |
| 2024年 | 18,000万元 | 1,350万元 | 营收16,800万，净利润1,180万 ✗ |

> **注**：2024年营收未达标（达成率93.3%），触发第5.2条股权补偿条款。

---

## 第5.2条 估值调整（股权补偿）

若2022年或2023年任一年度营业收入未达到上表承诺金额的80%，创始股东应无偿向投资方转让相当于增资完成时应持股本1.5%的公司股权。

**触发情况**：2024年度营收未达标，但协议约定触发年度为2022/2023年，故2024年未达标不触发本条款。创始股东与投资方另行协商补偿方案。

---

## 第5.3条 回购条款

若发生以下任一情形，投资方有权要求创始股东按投资本金加上年化8%的收益回购其持有的公司全部或部分股权：

| 触发情形 | 是否触发 | 说明 |
|---------|---------|------|
| 公司未能在2025年12月31日前完成合格IPO | 否 | 尚在辅导期 |
| 连续两个年度未达成业绩承诺 | 否 | 仅2024年未达标 |
| 创始股东严重违约或重大资产转让 | 否 | - |
| 公司核心业务发生重大不利变化 | 否 | - |

**回购价格计算**：
- 回购金额 = 投资本金 × (1 + 8% × 持有年数)
- 当前可回购金额上限 = 1,500 × (1 + 8% × 4) = 1,980万元

---

## 第5.4条 反稀释保护

若公司后续融资估值低于本轮估值（1.5亿元），投资方有权获得创始股东无偿配发股权以维持其持股比例。

---

## 第6条 重大事项否决权

投资方对以下事项享有一票否决权：

| 否决事项 | 条款 |
|---------|------|
| 增减注册资本 | ✓ |
| 变更公司主营业务 | ✓ |
| 超过1,000万元的对外投资或资产收购 | ✓ |
| 关联交易（单笔超过100万元） | ✓ |
| 知识产权（专利）转让或独家许可 | ✓ |
| 变更董事会构成 | ✓ |
| 核心管理层离任 | ✓ |

---

## 第7条 清算优先权

若公司发生清算、解散或被收购，投资方优先于其他股东按以下顺序分配：

1. 第一顺位：返还投资本金1,500万元
2. 第二顺位：年化8%的优先分配
3. 剩余资产：按持股比例分配
